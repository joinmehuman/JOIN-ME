JOIN ME
Concept How Value Message

答えを探すより、
自分と出会おう。

あなたは、一つの言葉では表せません。

JOIN MEは、あなたを分類したり、 決めつけたりする場所ではありません。

複数の視点や、あえて逆の可能性にも目を向けながら、あなた自身が自分を見つめ直すための場所です。

自己観察を始める

自己理解は、「答え」を見つけることではない。

「私はこういう人だ。」

そう思っていることもら環境や時間、人との関わりによって 変わるかもしれません。

一つの見方だけで自分を決めるのではなく、 別の可能性にも目を向ける。

JOIN MEは、そんな自己観察のきっかけをつくります。

3つのステップ

1

今の自分を言葉にする

感じていることや 考えていることを書き出します。

2

複数の視点を受け取る

AIは一つの答えではなく、 異なる解釈や反対の仮説も提示します。

3

自分で考える

最後に答えを決めるのはAIではありません。
あなた自身が、 納得できる答えを見つけます。

JOIN MEが大切にしていること

正解を決めない

一つの結論だけを押し付けません。

複数の可能性を見る

異なる視点を知ることで、思考の幅を広げます。

逆の発想も考える

今の答えとは反対の可能性にも目を向けます。

自分で選ぶ

答えはAIではなく、あなた自身が見つけます。

あなたは、
まだ知らない自分と出会える。

自分を知ることは、 自分を決めることではありません。

変わることも、 迷うことも、 矛盾することも、 あなたの一部です。

JOIN MEは、 答えを与えるサービスではなく、 新しい視点と問いを届けるサービスです。

今日の問いから始めよう。

今のあなたは、
本当に「そのあなた」だけでしょうか。
